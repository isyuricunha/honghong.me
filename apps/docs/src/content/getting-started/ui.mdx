## Prerequisites

- React 19
- Next.js 15
- Tailwind CSS
- TypeScript

## Installation

```bash
npm i @tszhong0411/utils cva@1.0.0-beta.4 lucide-react
# or
pnpm add @tszhong0411/utils cva@1.0.0-beta.4 lucide-react
# or
yarn add @tszhong0411/utils cva@1.0.0-beta.4 lucide-react
# or
bun add @tszhong0411/utils cva@1.0.0-beta.4 lucide-react
```

```css title='globals.css'
@import 'tailwindcss';
@import '@tszhong0411/ui/styles/preset.css';
@import '@tszhong0411/ui/styles/shiki.css'; /* Optional */
@import '@tszhong0411/ui/styles/typography.css'; /* Optional */
```

import { Button } from '@/components/ui/button'
