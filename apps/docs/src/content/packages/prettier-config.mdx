## Installation

You can install the prettier config package and its peer dependencies using your preferred package manager:

```bash
npm i @tszhong0411/prettier-config prettier-plugin-packagejson prettier-plugin-tailwindcss
# or
pnpm add @tszhong0411/prettier-config prettier-plugin-packagejson prettier-plugin-tailwindcss
# or
yarn add @tszhong0411/prettier-config prettier-plugin-packagejson prettier-plugin-tailwindcss
# or
bun add @tszhong0411/prettier-config prettier-plugin-packagejson prettier-plugin-tailwindcss
```
