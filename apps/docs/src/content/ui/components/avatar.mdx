<Callout title='Note'>
  You can use `getAbbreviation{:.entity.name.function}` to get the abbreviation of a name to be used in an `<AvatarFallback />{:tsx}`.
</Callout>

## Installation

```bash
pnpm dlx shadcn@latest add "https://docs.honghong.me/r/avatar"
```

## Usage

```tsx
import { Avatar, AvatarFallback, AvatarImage } from '@/components/ui/avatar'
```

```tsx
<Avatar>
  <AvatarImage src='https://github.com/tszhong0411.png' alt='@tszhong0411' />
  <AvatarFallback>TH</AvatarFallback>
</Avatar>
```
